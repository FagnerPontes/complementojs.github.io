1.1 Um pouco de história

A linguagem de programação JavaScript foi desenvolvida por Brendan Eich, um engenheiro da empresa Netscape no ano de 1995 e foi lançada em 1996. Seu nome original foi LiveScript e posteriormente foi renomeada para JavaScript. Deve-se ter cuidado para não confundir com a linguagem de programação java desenvolvida pela empresa Sun Microsystem. Atualmente a linguagem JavaScript pertence a Oracle assim como a linguagem java.

Desde então o JavaScript tem sido utilizado amplamente em vários segmentos no desenvolvimento de sistemas. A linguagem foi projetada para atuar em ambiente de terceiro, como um script no código fonte. Devido a esta característica o JavaScript não possui o conceito de entrada e saída e é de responsabilidade do ambiente promover mecanismos para a comunicação entre a linguagem e o exterior.

Os ambientes mais famosos onde existem aplicações do JavaScript são os Navegadores (Brownsers), como por exemplo; o Google Chrome, o FireFox e o Microsoft Edge. Entretanto, muitos outros ambientes utilizam interpretadores JavaScript como; banco de dados, editores de texto, editores de imagem, servidores, e muitos outros.

1.3 Definição de JavaScript

JavaScript é uma linguagem de programação scripting, ou seja, interpretada em um ambiente de terceiro. Também é uma linguagem multiparadigma, permitindo que a linguagem suporte mais de um paradigma de programação, podendo atuar como uma linguagem procedural ou como uma linguagem orientada a objetos.

Diferentemente de algumas linguagens de programação o JavaScript foi desenvolvido para rodar no lado cliente da web, por essa razão, a linguagem se tornou uma ferramenta poderosa e largamente utilizada para manipular o comportamento de páginas web de forma mais rápida e dinâmica, pois o código é interpretado diretamente no lado do cliente pelo Navegador. Outra característica importante é que o JavaScript é uma linguagem de tipagem dinâmica, isso implica dizer que os tipos de dados das variáveis não precisam ser declarados.

A linguagem JavaScript possui uma sintaxe similar à outras linguagens orientadas a objetos como o Java e o C++. Entretanto, diferentemente dessas linguagens, o JavaScript não possui o conceito de classe, sendo sua programação baseada em protótipos e com funções de primeira classe. A programação baseada em protótipos é um tipo específico de programação orientada a objetos onde o comportamento de herança é realizado através de um processo de reutilização de objetos existentes chamados de protótipos. Os protótipos são objetos generalizados que podem ser clonados e estendidos além de permitir que propriedades e métodos, de qualquer objeto, sejam adicionados dinamicamente. Funções de primeira classe são funções que podem ser manipulas da mesma forma que um objeto, sendo conhecidas por cidadãos de primeira classe ou objetos de primeira classe. Como qualquer objeto, elas podem ser passadas como argumentos, retornadas de uma função e atribuídas a uma variável.

Atenção!!!

Em 2015, foi introduzido na linguagem JavaScript o controlador class (classe). Entretanto as classes criadas em JavaScript são apenas simplificações para o modelo de herança por protótipo. O controlador class não introduz o conceito de classe atribuído às outras linguagens de programação. As classes podem ser entendidas como funções especiais.

Outra característica importante que não pode ser esquecida é que a linguagem JavaScript é case sensitive, isso significa que valores em caixa alta (maiúsculos) são diferentes de valores em caixa baixa (minúsculos). Dessa forma, comandos e declarações com o mesmo conteúdo, mas que possuam alguma diferença de caixa alta ou caixa baixa em seu corpo, serão tratados de forma diferente: var Nome é diferente de var nome e nome.GetSobrenome() é diferente de nome.getsobrenome().

Principais características do JavaScript:

  • Linguagem multiparadigma;
  • Funções são objetos e podem ser passadas como argumento;
  • Tipagem dinâmica, não há necessidade de declarar os tipos de dados das variáveis;
  • Linguagem Interpretada (não compilada).
  • Case sensitive.

1.5 Inserindo um script JavaScript em um arquivo HTML

Existem duas maneiras para programar com JavaScript em um documento HTML, a primeira é através de um script embutido diretamente no documento HTML (.html) entre suas tags. A segunda, é trabalhar com um script externo em um arquivo JavaScript (.js) e referenciar este arquivo no documento HTML. Veja a seguir um exemplo de cada uma dessas maneiras:

1.5.1 Script embutido no HTML

Abra o Visual Studio Code abra o menu File e em seguida no item Open File, Selecione a pasta onde você salvou o arquivo index.html, selecione o arquivo index.html, clique em Abrir.

No documento index.html escreva o seguinte código:

<!DOCTYPE html>

<html>

<head>

  <meta charset="utf-8">

</head>

<body>

  <script type="text/JavaScript">

    document.write("Olá, Mundo");

  </script>

</body>

</html>

Agora salve as alterações no documento e usando o Explorador de Arquivos abra a pasta onde está salvo o arquivo index.html e abra este arquivo através de um navegado. O navegador apresentará o seguinte resultado:

Olá, Mundo

O código anterior apresenta o método document.write("Olá Mundo"), esse método tem a função de escrever na página html a string atribuída em seu parâmetro no formato de texto.

Atenção!!!

Lembre-se sempre de salvar as alterações no documento, pois suas alterações só aparecerão no Navegador se você as salvar!

Todo código escrito em linguagem JavaScript deverá estar entre as tags <script></script>. Verifique que é necessário adicionar o campo (type="text/javascript"), pois é dessa forma que informamos ao interpretador do navegador que este trecho é um complemento JavaScript!

1.5.2 Script externo inserido por referência

Abra o Visual Studio Code, clique no menu File e em seguida no item Open File, Selecione a pasta onde você salvou o arquivo script.js, selecione o arquivo e clique em Abrir.

No documento script.js escreva o seguinte código:

document.write("Olá, Mundo");

Agora salve as alterações no documento.

No documento index.html escreva o seguinte código:

<!DOCTYPE html>

<html>

<head>

  <meta charset="utf-8">

  <script src="script.js" type="text/javascript"></script>

</head>

<body>

</body>

</html>

Agora salve as alterações no documento e usando o Explorador de Arquivos abra a pasta onde está salvo o arquivo index.html e abra este arquivo através de um navegador.

Através do campo (src="script.js) informamos qual arquivo queremos usar como referência no documento HTML, verifique que também precisamos informar o (type="text/javascript") para que o interpretador reconheça a linguagem JavaScript. Esses campos devem estar dentro da tag <script>.

1.6.2 Janela de Confirmação, método confirm()

O método confirm() é utilizado para validar se o usuário quer realizar ou não uma ação. Este método exibe uma janela onde o usuário será questionado sobre a realização de uma ação e terá que responder clicando nos botões Ok ou Cancelar. O botão Ok retorna verdadeiro (true) e o botão Cancelar retorna falso (false). O método confirm() pode ser declarado de duas maneiras:

window.confirm('texto');

confirm('texto');

Veja um exemplo de utilização do método confirm() onde será exibida uma janela de confirmação. Ao escolher uma das opções (Ok ou Cancelar) será exibido ao usuário o valor da variável de retorno do método confirm().

    document.write("sua opção foi :" + option); //Comando para exibir o resultado no navegador

  </script>

</body>

</html>

1.7 Variáveis

1.7.1 Variáveis Primitivas

Valores primitivos não podem ser representados por objetos e consequentemente não existem métodos relacionados a eles. Na prática, um valor primitivo é representado no nível mais baixo de implementação de uma linguagem. Em JavaScript existem 6 (seis) tipos primitivos:

  • string: Como em toda linguagem de programação são dados constituídos por um ou mais caractere comumente usados para representar textos, as strings devem ser declaradas entre aspas simples ('') ou aspas duplas ("");
  • number: É um tipo de dado numérico de dupla precisão em formato de ponto flutuante de 64 bits;
  • boolean: É um tipo de dado lógico que pode receber apenas um de dois valores possíveis: true (verdadeiro) ou false (falso);
  • null: Trata-se de um literal que representa um valor nulo;
  • undefined: Representa um dado que possui um valor indefinido, como uma variável que não teve um valor atribuído.
  • symbol: É um tipo de dado primitivo que armazena um único valor servindo como identificador único.

1.7.2 Variáveis Objetos

São variáveis independentes que podem possuir um conjunto de propriedades e métodos relacionados a elas. Existem 5 (cinco) formas de variáveis objetos em JavaScript:

  • String: São dados constituídos por um ou mais caractere comumente usados para representar textos, as strings devem ser declaradas entre aspas simples ('') ou aspas duplas ("");
  • Number: São dados numéricos sejam eles integer, float ou double;
  • Boolean: São os valores booleanos true (verdadeiro) e false (falso);
  • Array: São coleções de valores, que podem ser de qualquer tipo;
  • Funções: Em JavaScript funções são objetos, ou seja, elas podem ser declaradas como variáveis e utilizadas como parâmetro em outras funções.

1.7.3 Declaração das variáveis

Existem três maneiras de declarar variáveis em JavaScript:

  • var:  Declara uma variável de contexto global, independente do bloco de instrução (entre as chaves {}) onde foi criada, que possui atribuição dinâmica de escrita e leitura. Variáveis declaradas com var assumem contexto local se declaradas dentro de uma função.
  • let:  Declara uma variável de contexto local, só existem no escopo de bloco onde foram criadas (entre as chaves {}), que possui atribuição dinâmica de escrita e leitura;
  • const: Declara uma variável constante apenas de leitura.

A linguagem JavaScript possui a característica de ser dinamicamente tipada, por isso não é preciso identificar o tipo de dado. Os tipos são identificados automaticamente.

2.6 Funções em JavaScript

Funções são blocos de instrução responsáveis por procedimentos específicos realizados através de uma chamada que atende a uma necessidade da aplicação. Para declarar uma função se utiliza a palavra-chave function seguida por um nome e os parâmetros (argumentos) listados dentro dos parênteses e separados por vírgula, para finalizar se declara o procedimento (instruções) que será executado entre as chaves.

Sintaxe:

function myFunction(parâmetro) { código }

Exemplo:

function quadrado(x) { return x * x; }

Observação:

Quando o parâmetro é um dado primitivo, alterações realizadas no valor do parâmetro não são refletidas globalmente. Quando o parâmetro é um objeto as alterações realizadas nos atributos (propriedades) do objeto são refletidas globalmente.

Exemplo:

var a = 10; //variável primitiva

var b = { valor: 10 }; //objeto inicializado com o atributo valor = 10

function quadrado(x) {

  x = x * x;

  return x;

}

function cubo(x) {

  x.valor = x.valor * x.valor * x.valor;

  return x.valor;

}

var c = quadrado(a); //c = 100, a = 10

var d = cubo(b); //d = 1000, b.valor = 1000

2.6.1 Função lambda ou função flecha (=>).

A função lambda funciona da mesma forma que uma função callback (função de retorno). O operador representa uma forma simplificada de escrever uma função.

Sintaxe:

(parâmetro) => { código };

Os parênteses à esquerda são os parâmetros da função e o operador que está à direita é o corpo da função com expressão de retorno.

Exemplo:

var a = 10;

var b = (x) => { return x * x; }

UNIDADE 03 | Aplicar programação orientada a objetos com o JavaScript

Em JavaScript objetos são considerados como entidades independentes que possuem propriedades particulares e um tipo específico. Os objetos criados através da linguagem de programação podem ser comparados a objetos reais que possuem características responsáveis por defini-lo. As propriedades de um objeto definem suas características e podem ser entendidas como variáveis associadas a ele. Além dos objetos que já vimos na competência anterior (variáveis e funções), nós podemos criar e definir nossos próprios objetos.

A programação orientada a objetos é fundamenta em quatro grandes pilares:

  • Abstração: É o ponto de partida da criação de objetos. A abstração é responsável por extrair as características do objeto, atribuindo e relacionando valores, dando uma composição à sua estrutura. É através da abstração que definimos as entidades do mundo real.
  • Encapsulamento: Como seu próprio nome diz, o encapsulamento é a técnica utilizada para encapsular a informação, na prática, isso implica em esconder aquilo que não precisa ser exibido, atribuindo níveis de visibilidade a variáveis e métodos, como público ou privado.
  • Herança: É a técnica que permite criar uma classe a partir de outra que já existe, a classe recém-criada é conhecida como subclasse e contém todos os métodos e atributos da classe da qual herdou suas características (superclasse). A subclasse pode conter outras variáveis e métodos que não existem na superclasse. Como em JavaScript não existe o conceito de classe, a herança ocorre de maneira diferente das demais linguagens orientadas a objeto, sendo necessário a utilização de um mecanismo chamado protótipo.
  • Polimorfismo: É uma técnica que permite que um método ou função assuma formas diferentes para atender a demandas diferentes. As formas mais conhecidas de polimorfismos são:

›          Sobrescrita, consiste em sobrescrever um método que foi definido em uma superclasse através de herança (overriding)

›          Sobrecarga, onde uma função é redeclarada com outros parâmetros (overload). Ou quando usamos operadores que produzem resultados diferentes de acordo com o tipo de dado utilizado.

›          Parametrização, onde um tipo de dado ou função é escrita genericamente permitindo a manipulação de valores independentemente do tipo de dado.

3.1.3 Date

Objetos Date representam um valor temporal expresso em milissegundo. O Date possui os seguintes construtores:

  • new Date(); Criação de um objeto sem valor definido.
  • new Date(valor); valor  deve ser um número inteiro em milissegundo que representará uma data iniciando em 01 de janeiro de 1970.

›          Ex.: new Date(1655061600000); // Sun Jun 12 2022 16:20:00 GMT-0300

  • new Date(dataString); dataString deve ser uma String que representa uma data. A String deve possuir os seguintes formatos:

›          "Ano/Mês/Dia,Hora:Minutos:Segundos".

»        Ex.: new Date("2022/06/12,16:20:00"); // Sun Jun 12 2022 16:20:00 GMT-0300

»        Ex.: new Date("2022/06/12 "); // Sun Jun 12 2022 00:00:00 GMT-0300

›          "Ano,Mês,Dia,Hora:Minutos:Segundos".

»        Ex.: new Date("2022,06,12,16:20:00"); // Sun Jun 12 2022 16:20:00 GMT-0300

»        Ex.: new Date("2022,06,12"); // Sun Jun 12 2022 00:00:00 GMT-0300

  • new Date ("YYYY-MM-DDTHH:mm:ss.sssz").

›          (YYYY) representa o ano com 4 dígitos.

›          (-) separador entre os campos da data.

›          (MM) representa o mês 01 (janeiro) a 12 (dezembro).

›          (DD) representa o dia de 01 a 31.

›          (T) indica o início do elemento de tempo (horas).

›          (HH) Representa o tempo em horas com dois dígitos de 00 a 24.

›          (:) Separador do tempo em horas.

›          (mm) Representa o tempo em minutos com dois dígitos de 00 a 59.

›          (ss) Representa o tempo em segundos com dois dígitos de 00 a 59.

›          (.) Separador do campo em milissegundos.

›          (sss) é o número de milissegundos completos desde o início do segundo como três dígitos decimais.

›          (z) é o deslocamento de fuso horário especificado.

»        Ex: new Date("2022-06-12T16:20:00"); // Sun Jun 12 2022 16:20:00 GMT-0300

  • new Date(ano, mês, dia, hora, minuto, segundo, milissegundo); Diferentemente do construtor new Date(dataString), este construtor recebe valores numéricos, onde a contagem dos meses inicia no 0 e não no 1, dessa forma, o valor 0 representa janeiro e 11 representa dezembro.

›          Ex.: new Date(2022,05,12,16,20,00,000); // Sun Jun 12 2022 16:20:00 GMT-0300.

›          Ex.: new Date(2022,05,12); // Sun Jun 12 2022 16:20:00 GMT-0300.

Arrays são objetos globais de alto nível semelhantes às listas. Em JavaScript arrays possuem 3 tipos de construtores.

Principais Métodos:

Concatena o array selecionado com os arrays fornecidos no parâmetro.

Sintaxe:

array1.concat(array2, array3, ...);

Exemplo:

const array1 = ['1', '2'];

const array2 = ['3', '4'];

const array4 = array1.concat(array2, array3);

document.write(array4);

Itera o array em um laço através de uma função callback.

Sintaxe:

myArray.forEach(function (valorAtual, índice, array) { código }, value)

O parâmetro function (valorAtual, índice, array){código} é obrigatório e corresponde a função que será executada para iterar cada elemento. O parâmetro valorAtual é obrigatório e corresponde ao valor do elemento atual do array. O parâmetro índice é opcional e corresponde a posição do elemento atual. O parâmetro array é opcional e corresponde ao array do elemento atual. O parâmetro value é opcional e corresponde a um valor que se deseja passar para a função.

Exemplo:

var notas = [5, 6, 8, 4];

var ponto = 1;

var notaFinal = new Array(4);

notas.forEach(function (valorAtual, index, array) {

  notaFinal[index] = valorAtual + ponto;

});

document.write(notaFinal);

Resultado:

6,7,9,5

myArray.pop()

myArray.shift()

var notas = [5, 6, 8, 4];

var notaRemovida = notas.pop();

3.2 Criando um objeto em JavaScript

Existem três maneiras para criar objetos em JavaScript. A primeira é através de inicializadores de objetos, a segunda com funções construtoras e a terceira utilizando o método Object.create. Introduziremos também um novo conceito de laço de repetição: o for...in. Este laço é utilizado especificamente para percorrer todas as propriedades numeráveis de um objeto.

Sintaxe:

for (var i in objeto) { código_rotina };

3.2.1 Utilizando inicializadores de objetos:

Sintaxe:

var meuObjeto = {

  atributo_1: valor_1,

  atributo_2: valor_2,

  ...,

  atributo_n: valor_n,

}

Os atributos podem receber valores numéricos, strings, arrays, valores booleanos, funções e outros objetos. Veja o exemplo a seguir:

Personagem com os seguintes atributos:

  • Gêneros: Feminino ou Masculino
  • Habilidades: Arqueirismo ou Esgrima.
  • Espécies: Humano ou Elfo.
  • Equipamento: Se a habilidade for Arqueirismo será um Arco curto, se for Esgrima será uma Espada curta.

Exemplo:

<!DOCTYPE html>

<html>

<head>

  <meta charset="utf-8" />

</head>

<body>

  <script type="text/JavaScript">

    var Personagem = {

      Genero: 'Masculino',

      Habilidade: 'Arqueirismo',

      Especie: 'Elfo',

      Equipamento: function () {

        if (this.Habilidade == 'Arqueirismo') { return 'Arco curto'; }

        else if (this.Habilidade == 'Esgrima') { return 'Espada curta'; }

      }

    };

    for (var i in Personagem) {

      if (i == 'Equipamento') {//Equipamento é uma função. Deve-se usar () após a chamada

        document.write(i + " = " + Personagem.Equipamento() + "<br>");

        continue;

      }

      document.write(i + " = " + Personagem[i] + "<br>");

    }

  </script>

</body>

</html>

Habilidade = Arqueirismo
Especie = Elfo

O Exemplo anterior incluiu uma função para atribuir o valor da propriedade "Equipamento". Devido a este atributo ser uma função, sua chamada deverá ser finalizada com os parênteses (), diferentemente dos demais atributos. Unidade 04 | Conhecer os fundamentos do jQuery

O jQuery é uma biblioteca JavaScript que foi idealizada para facilitar a utilização da linguagem na produção de sites. O jQuery possui funções que foram construídas com o objetivo de simplificar a escrita, resumindo várias linhas de código em apenas uma linha de comando. As principais funções da biblioteca são:

  • Manipulação de contextos HTML.
  • Manipulação de contextos CSS.
  • Efeitos visuais.
  • Animações.

Antes de começar a usar o jQuery é necessário baixa a biblioteca no seu site oficial jquery.com. Existem duas possibilidades de download, uma versão compactada (compressed) e outra descompactada (uncompressed). Recomendo que você utilize a versão descompactada, pois esta versão apresenta uma melhor legibilidade.

4.2 Inserindo o arquivo jQuery no seu arquivo HTML

Inserimos um arquivo jQuery da mesma maneira que inserimos qualquer outro arquivo JavaScript (*.js). O local e nome do arquivo são declarados em (src), que deve estar declarado entre as tags (<script>).

Sintaxe:

<script src="jquery-3.7.1.js"></script>

4.3 Sintaxe básica de comandos jQuery

O jQuery utiliza uma estrutura de escrita baseada no CSS para realizar a seleção de elementos HTML através de seletores. A sintaxe básica é a seguinte: $(seletor).ação(), onde:

  • $ é usado para acessar a biblioteca jQuery.
  • (seletor) faz referência a elementos HTML.
  • ação() executa um método nos elementos HTML.

4.5 Manipulação de contextos HTML e CSS

Para evitar erros é recomendado que todos os métodos jQuery estejam escritos dentro do evento ready(). Este evento possui a seguinte sintaxe:

$(document).ready(function () {

  //métodos jQuery...

});

Desse modo, garantimos que os métodos jQuery sejam executados apenas quando o documento estiver completamente carregado.

4.5.1 Eventos

sintaxe:

$(seletor).on("actionName", value, function () { código })

O campo actionName é obrigatório e corresponde à uma string com o nome da ação a ser executada (click, dbclick, mouseenter, mouseleave, ...), value é opcional e corresponde a um objeto com valores que serão utilizados pela função disparada pelo evento e function é obrigatório, trata-se de uma função que será executada pelo evento.

click - Clique único com mouse

Sintaxe:

$(seletor).on("click", function () { código });

Exemplo:

$(document).ready(function () {

  $("p").on("click", function () {

    alert("você realizou o evento");

  });

});

4.5.2 Métodos

Os métodos disponíveis na biblioteca jQuery são responsáveis por realizar manipulações em elementos de acordo com a necessidade do desenvolvedor.

addClass(), removeClass()

addClass(): Insere novas classes ao elemento HTML selecionado.

removeClass(): Remove classes do elemento HTML selecionado.

Sintaxe:

$(seletor).addClass("nomeClasse1 ... nomeClasse");

$(seletor).removeClass("nomeClasse1 ... nomeClasseN");

Os nomes das classes devem ser separados por um espaço vazio. Os parâmetros nameClasse1 e nameClasseN são os nomes das classes editadas em CSS que se deseja atribuir ou  remover do elemento selecionado.

Exemplo:

$(document).ready(function () {

  $("p").addClass("pClass1 pClass2");

});

 

after(), before()

after(): Insere conteúdo após o elemento selecionado. O conteúdo estará fora do elemento.

before(): Insere conteúdo antes do elemento selecionado. O conteúdo estará fora do elemento.

Sintaxe:

$(seletor).after("<tag>conteúdo</tag>");

$(seletor).before("<tag>conteúdo <tag>");

O uso das tags dentro do parâmetro é opcional. Caso o conteúdo seja passado entre tags, o elemento será incluído na página html como um filho do elemento pai do seletor.

Exemplo:

$(document).ready(function () {

  $("p").after("<label>novo conteúdo </label>");

});

 

append(), prepend()

append(): Insere conteúdo no final do elemento. O conteúdo está dentro do elemento.

prepend(): Insere conteúdo no início do elemento. O conteúdo está dentro do elemento.

Sintaxe:

$(seletor).append("<tag>conteúdo</tag>");

$(seletor).prepend("<tag>conteúdo</tag>");

O uso das tags dentro do parâmetro é opcional. Caso o conteúdo seja passado entre tags, o elemento será incluído na página html como um filho do elemento definido no seletor.

Exemplo:

$(document).ready(function () {

  $("p").append("<br>novo conteúdo </br>");

});

 

attr(), removeAttr()

attr(): Define ou retorna atributos e valores do elemento HTML selecionado.

removeAttr(): Remove atributos do elemento HTML selecionado.

Sintaxe:

//Retorna o valor:

$(seletor).attr("atributo");

//Define um atributo e valor:

$(seletor).attr("atributo", valor);

//Define vários atributos e valores:

$(seletor).attr({ "atribute1": "value1", ..., "atributeN": "valueN" });

//Retorna e define atributo e valor através de uma função:

$(seletor).attr(atributo, function (índice, valorAutal) { código });

Os parâmetros índice e valorAtual são valores de retorno, não são definidos pelo desenvolvedor. Estes campos são utilizados caso o desenvolvedor queira o número da posição do elemento (índice) e o valor do atributo (valorAtual).

//Remove o atributo:

$(seletor).removeAttr("atributo");

//Remove um ou mais atributos do elemento selecionado:

$(seletor).removeAttr("atributo1 ... atributoN");

Os atributos devem ser separados por um espaço vazio.

Exemplo:

$(document).ready(function () {

  //Retorna o valor

  $("p").attr("id");

 

  //Define um atributo e valor

  $("button").attr("disabled", true);

 

  //Define vários atributos e valores

  $("p").attr({ "id": "pId", "class": "pClass" });

 

  //Retorna e define atributo e valor através de uma função

  $("p").attr("id", function (index, value) {

    return "newId"; //substitui o id do <p>.

  });

 

  // Remove o atributo:

  $("p").removeAttr("id");//remove atributo id do elemento HTML.

 

  // Remove um ou mais atributos do elemento selecionado.

  $("p").removeAttr("id class");//remove os atributos id e class.

});

 

clone()

Faz uma cópia do elemento HTML selecionado

Sintaxe:

$(seletor).clone(booleanValue);

O parâmetro deverá ser um valor booleano, true ou false. Este valor definirá se os eventos atribuídos ao elemento também deverão ser clonados. O valor false é o valor padrão, e determina que os eventos não devem ser clonados.

Exemplo:

$(document).ready(function () {

  $("p").clone().appendTo("body");

});

prop(), removeProp()

prop(): Define ou retorna propriedade e valor do elemento HTML selecionado

removeProp(): Remove uma propriedade definida pelo prop(). Este método não deve ser utilizado para remover propriedades como id, class, e style, para remover essas propriedades é necessário utilizar removeAttr().

Sintaxe:

  //Remove uma propriedade:

  $(seletor).removeProp("propriedade");

Exemplo:

$(document).ready(function () {

  //Retorna o valor

  $("p").prop("id");

 

  //Define uma propriedade e valor

  $("p").prop("name", "pName");

Através das funções de efeitos podemos fazer com que as páginas fiquem mais dinâmicas, interativas e visualmente mais atraentes para os usuários.

5.1 Efeitos

hide()

Oculta um elemento HTML, aplicando efeitos sobre o seu tamanho, largura e opacidade.

Sintaxe:

$(seletor).hide(speed, callback);

Os parâmetros são opcionais, speed representa a velocidade com a qual o efeito será realizado, callback é uma função que será executada após a conclusão do efeito, speed pode receber os valores: "slow", "fast" ou um número em milissegundos.

Exemplo: O código a seguir faz uma bolinha se tornar oculta:

<!DOCTYPE html>

<html>

<head>

  <style>

    .circulo {

      height: 100px;

      width: 100px;

      position: absolute;

      border-radius: 50%;

      background: rgb(10, 100, 150);

    }

  </style>

  <script src="jquery-3.7.1.js"></script>

  <script>

    $(document).ready(function () {

      $("button").on("click", function () {

        $("div").hide(1000);

      });

    });

  </script>

</head>

<body>

  <button>Aplicar Efeito</button>

  <br></br>

  <div class="circulo"></div>

</body>

</html>

fadeTo()

Realiza o desvanecimento de um elemento HTML até o nível de opacidade desejado.

Sintaxe:

Realiza o desvanecimento até uma opacidade com o valor entre 0 e 1 fornecido em opacity. A função callback é opcional e representa uma função que será executada após o elemento desvanecer.

Exemplo:

animate()

Realiza animações em elementos HTML através da manipulação de propriedades CSS. As propriedades devem estar no formato lowerCamelCase, ou seja, margin-left deve ser escrito como marginLeft. O lowerCamelCase é um formato em que as palavras são escritas juntas onde a primeira palavra deve ser escrita com letras minúsculas e as demais iniciando com a letra maiúscula. Não é possível realizar animações com cores.

Sintaxe:

$(document).ready(function () {

  $(seletor).animate({

    CssStyle1: 'valor1',

    CssStyle2: 'valor2',

    ...,

    CssStyleN: 'valorN'

  }, speed, callback);

});

Os parâmetros são opcionais, speed representa a velocidade com a qual o efeito será realizado, callback é uma função que será executada após a conclusão do efeito, speed pode receber os valores: "slow", "fast" ou um número em milissegundos.

Exemplo:

$(document).ready(function () {

  $("button").on("click", function () {

    $("div").animate({

      marginTop: '10px',

      opacity: '0.5',

      height: '50px',

    });

  });

});

 

stop()

Interrompe um efeito ou animação que está sendo executado no elemento selecionado.

Sintaxe:

$(seletor).stop();

Exemplo:

$(document).ready(function () {

  $("button").on("click", function () {

    $("div").stop();

  });

});

Aplicação em HTML:

Agora você irá realizar uma animação mais complexa, fazendo uma bolinha se mover para cima e para baixo simulando uma bola caindo e subindo, como se estivesse quicando em uma superfície. Além de realizar o movimento, a bolinha irá desvanecer até uma determinada transparência e achatar até uma determinada altura.

Para finalizar, será adicionado um botão para interromper a animação.

<!DOCTYPE html>

<html>

<head>

  <style>

    .circulo {

      height: 100px;

      width: 100px;

      position: absolute;

      border-radius: 50%;

      background: rgb(50, 100, 150);

    }

  </style>

  <script src="jquery-3.7.1.js"></script>

  <script>

    $(document).ready(function () {

      $("[name='start']").on("click", Descer);

      $("[name='stop']").on("click", PararEfeito);

    });

    function PararEfeito() {

      $(".circulo").stop();

      $("[name='stop']").attr("disabled", true);

      $("[name='start']").attr("disabled", false);

    }

    function Descer() {

      $("[name='stop']").attr("disabled", false);

      $("[name='start']").attr("disabled", true);

      $(".circulo").animate({ marginTop: '0px', opacity: '1.0', height: '100px' }, Subir);

    }

    function Subir() {

      $(".circulo").animate({ marginTop: '200px', opacity: '0.5', height: '90px' }, Descer);

    }

  </script>

</head>

<body>

  <button name="start">Aplicar Efeito</button>

  <button name="stop">Parar Efeito</button>

  <br></br>

  <div class="circulo"></div>

</body>

</html>
